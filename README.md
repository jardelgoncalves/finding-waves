## Run project
- Backend
  run command:
  ```
  npm i && npm run start:dev
  ```

- Frontend
  Add the URL of the API (for example: `http://localhost:3000`) and add the Google Maps token in `webapp/src/config.js` file

  And then run the command
  ```
  npm i && npm run start
  ```
